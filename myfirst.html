Container tag -

A container tag is a tag that has an opening and closing tag, and it can contain other HTML elements or text.
For example, the <div> tag is a container tag because it has both an opening <div> and a closing </div> tag.

paragraph tag - The <p> tag is used to define a paragraph in HTML. It is a block-level element that typically contains text content. Paragraphs are separated by vertical spacing, making the content more readable. The <p> tag can also contain other inline elements, such as links, images, and formatting tags like <strong> or <em>.
